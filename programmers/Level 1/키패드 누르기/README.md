# ✔️ 키패드 누르기
## 🌈 level 1
### 📌 2020 카카오 인턴십 코딩테스트

- 행렬 `arr[3][2]`
- 왼손 시작위치 `arr[3][0]` 왼손 사용 1,4,7
- 오른손 시작위치 `arr[3][2]` 오른손 사용 3,6,9
- 중앙 => 두손중 더가까운 2,5,8,0
- 거리가 같으면 왼손잡이는 왼손 사용 오른손잡이는 오른손 사용 / 거리는 1
- 왼손사용은 L 오른손 사용은 R
